최신 버전 v2.0.4

MoodLabs 설치 가이드

운영체제에 맞는 설치 스크립트를 다운로드해서 실행하면 ZIP 다운로드 · 압축 해제 · Chrome 열기까지 자동으로 처리됩니다. 마지막 2~3 클릭만 사용자가 직접 진행합니다.

macOS 설치

가장 쉬운 방법: Spotlight로 "터미널"을 열고 아래 한 줄을 붙여넣은 뒤 Enter.

curl -fsSL https://decordinated.github.io/moodlabs-ext-downloads/install-mac.command | bash

이 한 줄이면 Gatekeeper 경고 없이 바로 실행됩니다. (Homebrew/nvm 같은 도구도 쓰는 방식)

또는 파일 다운로드 후 직접 실행

⬇️ install-mac.command 다운로드
더블클릭하면 "Apple could not verify..." 경고가 뜹니다. 우회 방법:
  1. 다운로드한 파일을 더블클릭 (차단됨)
  2. 시스템 설정 → 개인정보 보호 및 보안으로 이동
  3. 맨 아래로 스크롤 → 차단된 파일 옆 "확인 없이 열기" 클릭
  4. 다시 더블클릭 → 이번엔 실행됨
또는 터미널에서 bash ~/Downloads/install-mac.command 한 줄 실행해도 됩니다.

스크립트가 자동으로 진행하는 작업

  1. 최신 버전 ZIP 다운로드 + SHA-256 검증
  2. 아래 경로에 압축 해제: ~/Library/Application Support/MoodLabs/extension
  3. 해당 폴더가 Finder에 열림 (경로는 클립보드에 복사됨)
  4. chrome://extensions/ 페이지가 자동 오픈

Chrome에서 마지막 단계 (첫 설치)

  1. 우상단 Developer mode 토글 켜기
  2. Load unpacked 버튼 클릭
  3. 다이얼로그에서 Cmd+Shift+G 누르고 Cmd+V로 경로 붙여넣기 → Enter, 또는 Finder에 열린 폴더를 직접 선택

업데이트 시

같은 스크립트를 다시 실행하면 새 버전으로 폴더가 덮어써집니다. Chrome에서 확장 카드의 ↻ 새로고침 아이콘만 한 번 누르면 최신 버전 적용 완료.

Windows 설치

가장 쉬운 방법: Win + R 눌러 실행 창 열기 → 아래 한 줄을 붙여넣고 Enter.

powershell -NoProfile -ExecutionPolicy Bypass -Command "iex (iwr 'https://decordinated.github.io/moodlabs-ext-downloads/install.ps1' -UseBasicParsing).Content"

SmartScreen 경고 없이 바로 실행됩니다. (PowerShell 직접 열어서 위 한 줄을 실행해도 동일)

또는 파일 다운로드 후 직접 실행

⬇️ install-windows.bat 다운로드
더블클릭하면 SmartScreen이 "Windows에서 PC를 보호했습니다"를 표시할 수 있습니다. "추가 정보" → "실행"을 눌러주세요.

스크립트가 자동으로 진행하는 작업

  1. 최신 버전 ZIP 다운로드 + SHA-256 검증
  2. 아래 경로에 압축 해제: %LOCALAPPDATA%\MoodLabs\extension
  3. 해당 폴더가 탐색기에 열림 (경로는 클립보드에 복사됨)
  4. chrome://extensions/ 페이지가 자동 오픈

Chrome에서 마지막 단계 (첫 설치)

  1. 우상단 Developer mode 토글 켜기
  2. Load unpacked 버튼 클릭
  3. 다이얼로그에서 주소창에 Ctrl+V로 경로 붙여넣기 → Enter, 또는 탐색기에 열린 폴더를 직접 선택

업데이트 시

같은 스크립트를 다시 실행하면 새 버전으로 폴더가 덮어써집니다. Chrome에서 확장 카드의 ↻ 새로고침 아이콘만 한 번 누르면 최신 버전 적용 완료.

문제가 생기면

스크립트가 막힌다면 버전별 ZIP 다운로드 페이지에서 직접 받아 수동 설치할 수 있습니다.
