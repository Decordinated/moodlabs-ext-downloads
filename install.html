최신 버전 v2.0.4

MoodLabs 설치 가이드

운영체제에 맞는 설치 스크립트를 다운로드해서 실행하면 ZIP 다운로드 · 압축 해제 · Chrome 열기까지 자동으로 처리됩니다. 마지막 2~3 클릭만 사용자가 직접 진행합니다.

macOS 설치

가장 쉬운 방법: Spotlight로 "터미널"을 열고 아래 한 줄을 붙여넣은 뒤 Enter.

curl -fsSL https://decordinated.github.io/moodlabs-ext-downloads/install-mac.command | bash

이 한 줄이면 Gatekeeper 경고 없이 바로 실행됩니다. (Homebrew/nvm 같은 도구도 쓰는 방식)

Chrome에서 마지막 단계 (첫 설치)

스크립트 실행이 끝나면 Chrome의 chrome://extensions/ 페이지가 자동으로 열립니다. 거기서 아래 3단계만 진행하면 끝.

1

우상단 "Developer mode" 토글 켜기

chrome://extensions 페이지의 오른쪽 위에 작은 토글 스위치가 있습니다. 회색이면 꺼진 상태, 파란색이면 켜진 상태. 한 번만 켜두면 다음부턴 안 건드려도 됩니다.

▲ chrome://extensions 우상단
2

"Load unpacked" 버튼 클릭

Developer mode를 켜면 페이지 좌상단에 Load unpacked / Pack extension / Update 세 버튼이 나타납니다. 첫 번째 Load unpacked를 클릭하세요.

▲ 좌상단 첫 번째 버튼
3

폴더 선택

파일 선택 다이얼로그가 열립니다. 경로는 이미 클립보드에 복사되어 있으니 붙여넣기가 가장 빠릅니다.

  1. Cmd + Shift + G 눌러 "폴더로 이동" 입력창 열기
  2. Cmd + V 로 경로 붙여넣고 Enter
  3. "열기" 버튼 클릭

또는 Finder에 자동으로 열린 extension 폴더를 직접 클릭해서 선택해도 됩니다.

업데이트 시
맨 위의 한 줄 명령어를 다시 실행하기만 하면 됩니다. 새 버전이 같은 폴더에 덮어써지고, Chrome 확장 카드의 ↻ 새로고침 아이콘 한 번만 누르면 끝.

Windows 설치

가장 쉬운 방법: Win + R 눌러 실행 창 열기 → 아래 한 줄을 붙여넣고 Enter.

powershell -NoProfile -ExecutionPolicy Bypass -Command "iex (iwr 'https://decordinated.github.io/moodlabs-ext-downloads/install.ps1' -UseBasicParsing).Content"

SmartScreen 경고 없이 바로 실행됩니다. (PowerShell 직접 열어서 위 한 줄을 실행해도 동일)

Chrome에서 마지막 단계 (첫 설치)

스크립트 실행이 끝나면 Chrome의 chrome://extensions/ 페이지가 자동으로 열립니다. 거기서 아래 3단계만 진행하면 끝.

1

우상단 "개발자 모드" 토글 켜기

chrome://extensions 페이지의 오른쪽 위에 작은 토글 스위치가 있습니다. 회색이면 꺼진 상태, 파란색이면 켜진 상태. 한 번만 켜두면 다음부턴 안 건드려도 됩니다.

▲ chrome://extensions 우상단
2

"압축해제된 확장 프로그램을 로드합니다" 버튼 클릭

개발자 모드를 켜면 페이지 좌상단에 세 버튼이 나타납니다. 첫 번째 "압축해제된 확장 프로그램을 로드합니다"(영문 Chrome에서는 Load unpacked)를 클릭하세요.

▲ 좌상단 첫 번째 버튼
3

폴더 선택

파일 탐색기 다이얼로그가 열립니다. 경로는 이미 클립보드에 복사되어 있으니 붙여넣기가 가장 빠릅니다.

  1. 다이얼로그 상단 주소 입력란을 클릭 (없으면 폴더 트리 위쪽 빈 공간 클릭)
  2. Ctrl + V 로 경로 붙여넣고 Enter
  3. "폴더 선택" 버튼 클릭

또는 탐색기에 자동으로 열린 extension 폴더를 직접 선택해도 됩니다.

업데이트 시
맨 위의 Win+R 한 줄 명령어를 다시 실행하기만 하면 됩니다. 새 버전이 같은 폴더에 덮어써지고, Chrome 확장 카드의 ↻ 새로고침 아이콘 한 번만 누르면 끝.

문제가 생기면

스크립트가 막힌다면 버전별 ZIP 다운로드 페이지에서 직접 받아 수동 설치할 수 있습니다.
