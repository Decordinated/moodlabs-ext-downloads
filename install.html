최신 버전 v2.0.4

MoodLabs 설치 가이드

운영체제에 맞는 설치 스크립트를 다운로드해서 실행하면 ZIP 다운로드 · 압축 해제 · Chrome 열기까지 자동으로 처리됩니다. 마지막 2~3 클릭만 사용자가 직접 진행합니다.

macOS 설치

⬇️ install-mac.command 다운로드

1. 다운로드한 파일 실행

Finder에서 install-mac.command 파일을 우클릭 → 열기를 눌러주세요. (첫 실행 시 macOS 보안 정책 때문에 더블클릭은 차단됩니다.)

"확인되지 않은 개발자" 경고가 뜨면 "열기"를 다시 눌러주세요. 시스템 설정 > 개인정보 & 보안 > "확인 없이 열기" 버튼을 눌러야 할 수도 있습니다.

2. 자동으로 진행되는 작업

  1. 최신 버전 ZIP 다운로드 + SHA-256 검증
  2. 아래 경로에 압축 해제: ~/Library/Application Support/MoodLabs/extension
  3. 해당 폴더가 Finder에 열림 (경로는 클립보드에 복사됨)
  4. chrome://extensions/ 페이지가 자동 오픈

3. Chrome에서 마지막 단계 (첫 설치)

  1. 우상단 Developer mode 토글 켜기
  2. Load unpacked 버튼 클릭
  3. 다이얼로그에서 Cmd+Shift+G 누르고 Cmd+V로 경로 붙여넣기 → Enter, 또는 Finder에 열린 폴더를 직접 선택

업데이트 시

같은 스크립트를 다시 실행하면 새 버전으로 폴더가 덮어써집니다. Chrome에서 확장 카드의 ↻ 새로고침 아이콘만 한 번 누르면 최신 버전 적용 완료.

Windows 설치

⬇️ install-windows.bat 다운로드

1. 다운로드한 파일 실행

다운로드 폴더에서 install-windows.bat를 더블클릭하세요.

Windows SmartScreen이 "PC를 보호했습니다"를 표시하면 "추가 정보" → "실행"을 눌러주세요.

2. 자동으로 진행되는 작업

  1. 최신 버전 ZIP 다운로드 + SHA-256 검증
  2. 아래 경로에 압축 해제: %LOCALAPPDATA%\MoodLabs\extension
  3. 해당 폴더가 탐색기에 열림 (경로는 클립보드에 복사됨)
  4. chrome://extensions/ 페이지가 자동 오픈

3. Chrome에서 마지막 단계 (첫 설치)

  1. 우상단 Developer mode 토글 켜기
  2. Load unpacked 버튼 클릭
  3. 다이얼로그에서 주소창에 Ctrl+V로 경로 붙여넣기 → Enter, 또는 탐색기에 열린 폴더를 직접 선택

업데이트 시

같은 스크립트를 다시 실행하면 새 버전으로 폴더가 덮어써집니다. Chrome에서 확장 카드의 ↻ 새로고침 아이콘만 한 번 누르면 최신 버전 적용 완료.

문제가 생기면

스크립트가 막힌다면 버전별 ZIP 다운로드 페이지에서 직접 받아 수동 설치할 수 있습니다.
